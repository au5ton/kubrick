# HEVC experiment

## Dependencies
- `pip3 install -r requirements.txt`
- `optipng` in PATH, see: http://optipng.sourceforge.net/
    - Linux Mint repo: https://community.linuxmint.com/software/view/optipng
    - Debian repo: https://packages.debian.org/buster/optipng

## Usage

`./experiment.py --help`
