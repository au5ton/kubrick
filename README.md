# kubrick
ffmpeg scripts to prepare media for editing

##### Usage
Append `tools/` directory to active path (doesn't modify .bashrc):
- `. ./activate.sh`

Help menu:
- `foo.sh --help`
- `foo.sh -h`
- `foo.sh`

##### Dependencies
- `ffmpeg`
- `bash`
- `wget`
- `bc`: arbitrary precision numeric processing language
